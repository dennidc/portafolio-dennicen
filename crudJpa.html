Galería del sistema de gestión para una biblioteca

Aplicación de escritorio para gestionar libros en una biblioteca de manera local, donde se presenta la información de los libros mediante una tabla, asi como la ubicación del libro en la biblioteca.

Inicio

Página principal del sistema CRUD de libros

Alta de libro

Ventana del programa donde se ve el formulario para que el usuario de de alta un libro, mostrando los respectivos botones de "Vaciar formulario" y "Añadir"

Catálogo de libros

Ventana del programa donde se visualizan los libros dados de alta, de igual forma contiene los botones para eliminar y editar la información del libro seleccionado.

Modificar Libro

Ventana del programa que muestra la información registrada en la base de datos, para modificar los apartados.

Eliminar un libro

Vista del programa que lanza un mensaje de confirmación para eliminar un libro.

Vista de la tabla libros

Se visualiza en la imagen la forma en la que se almacena la información en la base de datos.
